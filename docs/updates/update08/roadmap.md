# Update08 Roadmap

## 目的
保存データ表示モードで型編成ナビを実用化し、型検索から新規部隊を作成して評価・履歴・グループ管理まで保持できるようにする。

## 完了内容
- 保存データ表示モードでは型候補一覧を所有済み武将・装備・登録済み名馬へ限定する。
- 全データ表示モードでは従来どおり全候補を対象にする。
- 型候補一覧から「この型で新規部隊」を作成し、既存部隊を上書きしない。
- 新規部隊には選択中の評価型ID・評価型名・マイメモを自動入力する。
- 編成結果に評価型ID、評価型名、トータルスコア、評価スコア、マイメモを保持する。
- 1部隊あたり最大10件の評価履歴を保持する。
- 1グループあたり最大12部隊、最大5グループを保持する。
